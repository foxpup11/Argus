# AgentScope 开发日志

## 项目概述

AgentScope 是一个桌面应用，帮助开发者可视化 AI Agent（如 Claude Code）对代码做了什么。

**核心价值**：
- 一眼看清 Agent 改了哪些文件
- 每个改动关联到具体的 tool call
- 自动评估改动风险等级
- 语法高亮的 Diff 预览

## 更新记录

### 2026-06-30 - BUG 修复

#### Diff 视图文件不存在问题
| 问题 | 修复 |
|------|------|
| 文件存在但显示"文件不存在" | 修复相对路径处理逻辑 |
| 相对路径无法正确解析 | 改进 Git 仓库查找逻辑 |

**根因分析**：
- Claude Code 返回的文件路径可能是相对路径（如 `frontend/i18n.js`）
- 原代码首先调用 `os.Stat(filePath)` 检查文件存在性
- 当工作目录不是 Git 仓库根目录时，`os.Stat` 会失败
- 导致显示"文件不存在"错误

**修复方案**：
- 优先处理绝对路径情况
- 对于相对路径，先从会话获取工作目录
- 然后在 Git 仓库中查找文件
- 确保路径转换正确
