# AgentScope 开发日志

## 项目概述

AgentScope 是一个桌面应用，帮助开发者可视化 AI Agent（如 Claude Code）对代码做了什么。

**核心价值**：
- 一眼看清 Agent 改了哪些文件
- 每个改动关联到具体的 tool call
- 自动评估改动风险等级
- 语法高亮的 Diff 预览

## 技术架构

```
┌─────────────────────────────────────────────────────────────┐
│                    AgentScope Desktop                        │
├─────────────────────────────────────────────────────────────┤
│  Frontend (HTML + CSS + JS)                                  │
│  ├── index.html        # 主页面                              │
│  ├── style.css         # Apple HIG 风格样式                   │
│  ├── app.js            # 主逻辑                              │
│  ├── i18n.js           # 国际化                              │
│  └── resizer.js        # 拖拽调整宽度                        │
├─────────────────────────────────────────────────────────────┤
│  Backend (Go + Wails)                                        │
│  ├── main.go           # Wails 入口                          │
│  ├── app.go            # API 层                              │
│  │   ├── GetSessions() # 获取会话列表                        │
│  │   ├── GetSession()  # 获取会话详情                        │
│  │   └── GetDiff()     # 获取文件 Diff                       │
├─────────────────────────────────────────────────────────────┤
│  Core Engine (internal/)                                     │
│  ├── session/          # 会话解析                            │
│  │   ├── session.go    # 数据模型                            │
│  │   ├── discover.go   # 会话发现                            │
│  │   └── claude/       # Claude Code 解析器                  │
│  ├── diff/             # Git Diff 引擎                       │
│  │   ├── engine.go     # Git 操作封装                        │
│  │   └── matcher.go    # 文件-动作匹配                       │
│  └── risk/             # 风险规则引擎                        │
│      └── engine.go     # 规则评估                            │
└─────────────────────────────────────────────────────────────┘
```

## 技术栈

| 层级 | 技术 | 用途 |
|------|------|------|
